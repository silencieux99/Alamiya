# Assets nécessaires

Ce dossier doit contenir les fichiers suivants :

## Icônes et images

- `icon.png` (1024x1024) - Icône de l'application
- `adaptive-icon.png` (1024x1024) - Icône adaptative Android
- `splash.png` (1242x2436) - Écran de démarrage
- `favicon.png` (48x48) - Favicon web
- `notification-icon.png` (96x96) - Icône de notification

## Génération des assets

Vous pouvez utiliser des outils en ligne ou créer ces assets manuellement :

1. **Icon Generator** : [AppIcon.co](https://www.appicon.co/) ou [IconKitchen](https://icon.kitchen/)
2. **Splash Screen** : Créer un design simple avec le logo et le nom "Alamiya"
3. **Couleurs recommandées** :
   - Couleur principale : `#FF6B35`
   - Fond : `#FFFFFF`

## Placeholders temporaires

En attendant de créer les assets finaux, vous pouvez utiliser des placeholders ou générer des images simples avec un outil de design.
